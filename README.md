# Generative Modelling Case Study – GAN Assignment

## Overview

This repository contains my coursework on **Generative Adversarial Networks (GANs)**.  
The project explores GAN performance on both synthetic datasets and real-world datasets using PyTorch.

The assignment is divided into two parts:

1. **Building GANs from scratch on 2D synthetic data**
2. **Applying GANs to practical datasets**

---

## Contents

### Part 1: GANs on Synthetic Data

- Sine-wave dataset
- Spiral dataset
- Baseline GAN vs Modified GAN comparison
- Training loss curves
- Visual comparison of generated samples

### Part 2: Real-World GAN Applications

#### 2.1 BloodMNIST

- Blood cell microscope image generation
- DCGAN model
- Training progress and generated outputs

#### 2.2 CICIDS 2017

- Synthetic cybersecurity traffic generation
- Fully connected GAN
- PCA / t-SNE comparison
- Feature distribution analysis

#### 2.3 QuickDraw Pizza

- Pizza sketch generation
- DCGAN model
- Early vs final generated samples
